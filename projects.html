My Programming Projects

  1. U.S-Highway-Identifier - This is a Java project where the user inputs a U.S interstate highway number and the program returns whether the interstate is a primary or auxiliary highway. It also indicates if the (primary) highway runs north/south or east/west.
  2. Leet-speak-password-Generator - This Java program prompts the user to enter a simple lowercase password ( 8 - 15 characters in length ) and makes it stronger by making it leet.
  3. Decimation-Simulation - This Java program simulates the decimation of a population. The user enters the size of a population and a percentage to use for decimation.
  4. Radio Creator - This Java project is a multi-class program that creates two radio objects each with its own AM/FM band, volume, and frequency. The program adjusts these data members and prints out each radio's attributes.
